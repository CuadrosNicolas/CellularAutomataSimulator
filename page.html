Aide du simulateur
Le simulateur est compose de plusieurs partie :

  • Menu de generation de la grille, choisissez votre type d'automate, la taille hoziontale et verticale de la grille puis appuye sur "Generer" pour generer la grille
  • grille, la grille permet de visualiser la simulation, le clic gauche de la souris permet d'interagir avec en changeant la case selectionne par celle choisit dans les options de gestions
  • option de gestion de la simulation, ce menu vous permet de gerer :
    • La vitesse de la simulation en seconde
    • De generer les cases de la grille de facon aleatoire en fonction du seuil
    • D'effacer les cases de la grille
    • De passer "X" generation de la simulation pour atteindre une generation voulue
    • De choisir le type de case qu'impose le clic gauche de la souris sur la grille
    • Une description de l'automate choisit, son nom, ses regles et un lien plus de precision

Aide Script :
Le module de script vous permet de creer vos propre automate en utilsant le langage javascript (plus d'information sur ce langage)
Le script doit contenir :

  • Une fonction de mise jour de l'automate
  • Une fonction de generation aleaoire de la grille pour votre automate
  • Une definition de ses differents etats (nom,couleur,valeur)
  • Le nom de votre automate
  • Une description
